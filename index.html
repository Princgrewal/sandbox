Make a Form

Make your own form editing the form in the file myform.html

While running 'python server.py' from the command prompt, you can access myform.html with the url: http://localhost:8000/myform

Change the title, and add your own form elements to the form. Pick a theme such as sports, movies, TV, monsters, books, etc. Something that you'd like to enter information about.

Use form elements such as <input> (text, checkbox, radio), <label>, <fieldset>, <select>, <textarea>, and try adding some form elements that we haven't covered in class.

You can also check out my example Monster Form.

Look up details about form elements and try out other form elements found here:

  • Basic Native Form Controls
  • The HTML5 Input Types
  • Other Form Controls
  • Form Element Reference
  • The <input> Form Element

When you are done editing your form, while running 'python server.py' from the command prompt go to http://localhost:8000/myform, test your form and submit it with the submit button.
